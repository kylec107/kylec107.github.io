Evade Game

Created wih Java

I created this game by drawing from interpretations of the snake game and putting my own take on them. In this game the player controls their avatar (a white square) using the WASD keys and is shown the health bar, score, and level in real time. The goal of the game is to survive multiple waves of enemies varying in speed and intelligence by dodging the enemy avatars and eating apples to regenerate health. This game uses Object Oriented Programming to spawn enemies of various types at certain levels (i.e. Basic enemies, Fast enemeies who move faster, and Smart enemies who follow the player at a slow pace).
